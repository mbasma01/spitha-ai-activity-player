Translating a simple question into a powerful filter (both numeric and text).

Core Concept: The 3-Step Filtering Machine

Remember the three steps: 1. Ask a question (Create Mask), 2. Check each row, 3. Keep only the True rows.

C. Interactive Debugger (Fix Syntax)

Find the error in the filtering code and type the correct version below it.

Challenge 1: Equals vs. Assignment

We want to check if Genre is 'Action'. What's wrong with the operator?

movies_df[movies_df['Genre'] = 'Action']

Challenge 2: The Missing Wrapper

This code creates a True/False list, but doesn't actually filter the DataFrame. Wrap it!

movies_df['Release_Year'] > 2000

Challenge 3: Text requires Quotes

We are looking for the 'Drama' genre. Python thinks Drama is a variable name here.

movies_df[movies_df['Genre'] == Drama]
