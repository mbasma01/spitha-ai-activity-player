Method 1: Loading Data via a Raw GitHub Link

This method is the best way to load data from a reliable source like our course files. It's the most stable for running code.

1 Get the Raw GitHub URL

The necessary resource link (the raw GitHub URL) will always be provided to you directly within the current course exercise. If you ever need to find it yourself: when viewing a data file (like a CSV) in a GitHub repository, click the "Raw" button. This gives you the direct, plain-text link Colab needs.

The raw URL will always start with https://raw.githubusercontent.com/. You will paste this link directly into the code cell.

2 Use Python Code to Fetch the Data

In a new Colab code cell, we first ask Python to grab the powerful Pandas tool (which we call pd for short). Then we tell Pandas to read the data directly from the web link.

# Step 1: Import Pandas (the data analysis tool)
import pandas as pd

# Step 2: Define the URL (You will paste the link here)
github_url = "https://raw.githubusercontent.com/user/repo/main/data.csv"

# Step 3: Tell Pandas to read the data from that link
try:
    df = pd.read_csv(github_url)
    print("Data loaded successfully! Here are the first few rows:")
    print(df.head())
except Exception as e:
    print(f"Error loading data: {e}")

Method 2: Uploading a Local File

This method is easy for files saved on your computer, but be warned: the file is deleted when your Colab session ends or times out!

1 Click the Upload Button

In the left vertical sidebar of Google Colab, click the Files icon (1). Then, click the Upload icon (2) and select your file from your computer.

The Files icon (1) is the folder icon in the vertical toolbar. The Upload icon (2) is a sheet of paper with an upward arrow, located at the top of the file panel.

2 Reference the Local Path

Once uploaded, the file is available for Python to read. Right-click the file name in the sidebar and select "Copy path" to get the exact file name.

# Step 1: Import Pandas (the data analysis tool)
import pandas as pd

# Step 2: Define the local file path (Use the name of the file you uploaded)
file_path = "sales_data.csv"

# Step 3: Tell Pandas to read the data from the local file
try:
    df = pd.read_csv(file_path)
    print("Data loaded successfully! Here are the first few rows:")
    print(df.head())
except Exception as e:
    print(f"Error loading data: {e}")
