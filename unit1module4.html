üí° Analogy: The Storage Box (df)

In programming, we put things into variables (like a box) to hold them. When we read data, we need a special box called a DataFrame (df for short) to store the neat table. The assignment operator (=) says: "Take the data on the right, and put it into the box on the left."

What is a CSV?

A .csv (Comma-Separated Values) file is the simplest, most common way to store a spreadsheet. It's just a plain text file where every piece of data in a column is separated by a comma. Pandas is an expert at reading these files.
