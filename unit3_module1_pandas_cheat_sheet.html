Pro Tip

Always use inplace=True if you want to modify the dataframe directly without assigning it to a new variable.

Inspection

View first 5 rows
df.head()
Check types & missing counts
df.info()

Cleaning Nulls

Drop rows with missing values
df.dropna(inplace=True)
Fill missing values with Average (Mean)
mean_val = df['column_name'].mean()
df['column_name'].fillna(mean_val, inplace=True)

Transformation

Convert text to Number (Integer)
df['id'] = df['id'].astype(int)
Fix Typos (Replace Map)
df['gender'] = df['gender'].replace({'mle': 'Male', 'femal': 'Female'})

Filtering Outliers

Keep only logical values
# Keep Age between 0 and 100
df = df[(df['age'] >= 0) & (df['age'] <= 100)]
