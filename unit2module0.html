1. The Digital Catalog: Revisiting DataFrames

Before we filter, let's understand the structure and tools of Pandas.

A. Dataset Setup & Code Breakdown

Download Movies CSV 
import pandas as pd
movies_df = pd.read_csv('https://raw.githubusercontent.com/mbasma01/spitha-ai-notebooks/refs/heads/main/movies_data_sample.csv')
print(movies_df)

Code Breakdown:

  • import pandas as pd: This makes the **Pandas tools** available in your code, giving it the common nickname `pd`.
  • movies_df = pd.read_csv(...): This is the critical line! The `pd.read_csv()` function **fetches the data, processes it, and officially creates the DataFrame object**, which we name `movies_df`.

E. Understanding `.dtypes` and Data Types

**Purpose:** The `.dtypes` property tells you the *format* of the data in each column. This is essential because text (Object) requires `==` for matching, while numbers (Int/Float) allow comparison operators like `>` or `<`.
