5. Data Types: The Atoms of Data

Before we build tables, we need to understand the raw materials inside the cells.

A. The 4 Main Ingredients

Just like the universe is made of atoms, every piece of data in Pandas is made of a specific Type. Pandas behaves differently depending on the type.

Integer (int) int64

Whole numbers. No decimals.

Release Year: 2023
Votes: 1500
Float (float) float64

Numbers with decimals. Math-ready.

Rating: 7.8
Price: $12.99
String (str) object

Text. Always wrapped in quotes.

Title: "Avatar"
Genre: "Action"
Boolean (bool) bool

Logic switches. Only True or False.

is_sequel: True
has_oscar: False
