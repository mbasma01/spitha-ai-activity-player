API Reference

Pandas / Seaborn v0.12

Data Loading

pd.read_csv()

Load comma-separated values.

Distribution

sns.histplot()

Univariate. Shows frequency/count of one numerical variable.

sns.boxplot()

Summary. Shows median, quartiles, and outliers.

Relationships

sns.scatterplot()

Correlation. Shows relationship between two numbers.

sns.heatmap()

Matrix. Shows correlation strength across all numerical columns.

Categorical / Trends

sns.barplot()

Comparison. Shows aggregate (mean/sum) of value by category.

sns.lineplot()

Trend. Shows data evolution over time.
